# libusoft-mazescape

Game developed in Unity for the Libusoft Code Challenge. 
The original repo is: https://github.com/vedranserences/libusoft-cc

Game controls: W,S,A,D - movement

In easy game mode: 50 purple floor markings for orientation, you mark with F or E.
In medium game mode: 10 purple floor markings for orientation, you mark with F or E.
In hard game mode: 0 purple floor markings.
